## 🚀 Sobre

Este é o código-fonte do meu portfólio profissional, desenvolvido com HTML5, CSS3 e JavaScript vanilla.

**Destaques:**
- Design responsivo e moderno (dark theme)
- Animações suaves com Intersection Observer
- Código limpo e semântico
- Performance otimizada
- 100% estático (sem dependências de backend)

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Estilização:** CSS Variables, Flexbox, Grid, Animations
- **Fontes:** Inter, JetBrains Mono (Google Fonts)
- **Ícones:** Font Awesome 6
- **Hospedagem:** GitHub Pages
